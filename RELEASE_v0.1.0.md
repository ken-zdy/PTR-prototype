# PTR-prototype v0.1.0

## 发布摘要
本版本新增了面向原型评审与协同修改的留言系统，支持在页面任意位置创建留言点并进行线程化沟通。

## 主要更新
- 新增全局留言模式，可在任意页面位置点击创建留言点
- 新增线程式留言与回复，支持在同一个留言点持续讨论
- 新增留言状态管理，支持未解决/已解决切换
- 新增留言筛选视图：全部、未解决、已解决
- 新增留言点拖拽定位，支持调整留言点位置
- 新增快捷交互：快捷键 C 开关留言模式、下一条未解决快速跳转
- 新增留言人信息配置（姓名/角色），并在消息中展示留言人信息
- 支持本地持久化存储，并兼容旧版留言数据结构
